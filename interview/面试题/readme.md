4. 回流(重排)和重绘
DOM树: 表示页面结构(纯html),
渲染树:(DOM + CSSDOM) 表示DOM结点如何显示,

GPU拿到渲染树后, 进行布局绘制叫做重排
当页面上的DOM结点发生非布局变更时GPU需要再次绘制--重绘

怎样触发重排
- 页面首次渲染
- 添加/删除可见的dom元素
- 改变元素位置
- 改变元素尺寸
- 改变窗口大小
怎样触发重绘
- 当页面的dom改变了非几何信息的属性时
渲染树(GPU能读得的东西)
